# Window AI Task Package

Этот пакет содержит структурированную пользовательскую задачу для AI-анализа.

Если в пакете есть ai_index.md, начните с него. Он содержит быстрый маршрут и прямые ссылки на важные материалы.

## Как обработать пакет

1. Сначала прочитайте ai_index.md, если файл есть.
2. Затем прочитайте task/task.md.
3. Затем прочитайте manifest.json, чтобы понять состав пакета.
4. Изучите исходные файлы в files/.
5. Изучите скриншоты, связанные с файлами, в linked-screenshots/.
6. Изучите самостоятельный визуальный контекст в standalone-screenshots/.
7. Для каждого скриншота прочитайте metadata.json.
8. Для каждой аннотации используйте комментарий пользователя, crop-изображение и координаты на исходном скриншоте.

## Назначение папок

* files/: исходные загруженные файлы.
* linked-screenshots/: скриншоты, связанные с конкретным исходным файлом.
* standalone-screenshots/: самостоятельный визуальный контекст.
* crops/: выделенные пользователем фрагменты изображений.
