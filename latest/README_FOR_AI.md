# Window AI Task Package

Этот архив содержит структурированную пользовательскую задачу для AI-анализа.

## Как обработать пакет

1. Сначала прочитайте task/task.md.
2. Затем прочитайте manifest.json, чтобы понять состав пакета.
3. Изучите исходные файлы в files/.
4. Изучите скриншоты, связанные с файлами, в linked-screenshots/.
5. Изучите самостоятельный визуальный контекст в standalone-screenshots/.
6. Для каждого скриншота прочитайте metadata.json.
7. Для каждой аннотации используйте комментарий пользователя, crop-изображение и координаты на исходном скриншоте.

## Назначение папок

* files/: исходные загруженные файлы.
* linked-screenshots/: скриншоты, связанные с конкретным исходным файлом.
* standalone-screenshots/: самостоятельный визуальный контекст.
* crops/: выделенные пользователем фрагменты изображений.
